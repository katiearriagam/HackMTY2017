Description

Developed an application for iOS to help users with visual impairment get a better context of their surroundings. The application leverages an image recognition API to process pictures and provide information and text on found objects through voice feedback, as well as the UI. User interaction is enabled to work with Voice Over and receive voice commands to build a friendly user experience for the target audience.
